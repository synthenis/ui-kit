Getting Started
Installation
Synthenis UI Kit requires no build tools or dependencies. Add the stylesheet via CDN or clone the repository directly — then start building.
CDN
git clone
<!-- Add to <head> --> <link rel="stylesheet" href="https://cdn.synthenis.com/ui-kit/v1/synthenis-ui.min.css"> <!-- Optional: preconnect for faster font load --> <link rel="preconnect" href="https://cdn.synthenis.com">
$ git clone https://github.com/synthenis/ui-kit.git $ cd ui-kit $ open index.html
